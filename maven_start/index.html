Maven-start

作用

提供标准化的项目结构，构建流程，依赖管理机制。

项目结构

仓库

本地仓库 -> 远程仓库 -> 中央仓库

生命周期

指一次构建过程经历了多少个事件

一共3套生命周期：

-clean:清理工作

-default：核心工作，比如编译，打包等

-site: 产生报告，发布站点等

同一生命周期内，执行后续命令，前边的所有命令会自动执行
